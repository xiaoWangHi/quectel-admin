

### 安装



```bash

# 克隆项目
git clone https://github.com/xiaoWangHi/quectel-admin.git

# 进入项目目录
cd quectel-admin

# 安装依赖
npm install

# 本地开发 启动项目
npm run serve

# 打包项目
npm run build

```
----------


**项目文档**

> 本文档项目 基于 [vuepress](https://www.vuepress.cn/) 开发。


```bash

# 运行项目文档
npm run docs:dev

# 打包项目文档
npm run docs:build

```
